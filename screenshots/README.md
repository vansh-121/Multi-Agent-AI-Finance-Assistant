## Screenshot Guidelines

When updating or adding new screenshots:

1. **Resolution:** Use 1920x1080 or higher for clarity
2. **Format:** PNG for best quality
3. **Content:** Show real data examples with diverse stocks
4. **Privacy:** Remove any sensitive API keys or personal information
5. **Naming:** Use descriptive, lowercase names with underscores

## Taking New Screenshots

To capture current features:

```bash
# Start the application
streamlit run app.py

# Navigate to the feature you want to capture
# Use browser built-in screenshot or tools like:
# - Windows: Win + Shift + S
# - Mac: Cmd + Shift + 4
# - Linux: gnome-screenshot or spectacle
```
